**Gilt für**: Access 2013, Office 2013

Gibt den Namen des angegebenen Objekts zurück. Read-only **Zeichenfolge**.

## <a name="syntax"></a>Syntax

*Ausdruck* Eine Variable, die ein **Container-Objekt** darstellt.
