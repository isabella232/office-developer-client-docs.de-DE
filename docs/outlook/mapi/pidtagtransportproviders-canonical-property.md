# <a name="pidtagtransportproviders-canonical-property"></a>Kanonische Pidtagtransportproviders (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Liste der Bezeichner von Transportanbietern im aktuellen Profil.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_TRANSPORT_PROVIDERS  <br/> |
|Kennung:  <br/> |0x3D02  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |MAPI-Profil  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Verwenden Sie diese Eigenschaft nicht. Sie ist für die Verwendung durch MAPI reserviert.
