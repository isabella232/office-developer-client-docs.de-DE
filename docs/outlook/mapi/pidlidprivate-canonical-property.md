# <a name="pidlidprivate-canonical-property"></a>Kanonische PidLidPrivate-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob der Endbenutzer möchte, dass dieses Nachrichtenobjekt für andere Benutzer ausgeblendet wird, die Zugriff auf das Nachrichtenobjekt haben.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidPrivate  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008506  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Allgemeines Messaging  <br/> |
