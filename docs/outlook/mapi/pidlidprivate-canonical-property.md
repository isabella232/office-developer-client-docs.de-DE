# <a name="pidlidprivate-canonical-property"></a>Kanonische Pidlidprivate (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob der Endbenutzer dieses Nachrichtenobjekt für andere Benutzer, die Zugriff auf das Nachrichtenobjekt haben, ausgeblendet möchte.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidPrivate  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long-ID (Deckel):  <br/> |0x00008506  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Allgemeine Nachrichtenübermittlung  <br/> |
