# <a name="pidtagnondeliveryreportdiagcode-canonical-property"></a>Kanonische Pidtagnondeliveryreportdiagcode (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen Diagnosecode, der Teil eines Unzustellbarkeitsberichts ist.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_NDR_DIAG_CODE  <br/> |
|Kennung:  <br/> |0x0C05  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Allgemeine Nachrichtenübermittlung  <br/> |
