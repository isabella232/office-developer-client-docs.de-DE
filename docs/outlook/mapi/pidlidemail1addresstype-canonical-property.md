# <a name="pidlidemail1addresstype-canonical-property"></a>Kanonische PidLidEmail1AddressType-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Adresstyp der ersten E-Mail-Adresse an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidEmail1AddrType  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008082  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Wenn vorhanden, muss dieser Eigenschaftswert ein gültiger Adresstyp sein.
