# <a name="pidtagrpcoverhttpproxyprincipalname-canonical-property"></a>PidTagRpcOverHttpProxyPrincipalName (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Prinzipalnamen des Proxyservers, der von Microsoft Outlook zum Herstellen einer Verbindung mit Microsoft Exchange Server mithilfe eines Remoteprozeduraufrufs (RPC) über hypertext Transfer Protocol (HTTP) verwendet wird.
  
## 

|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_ROH_PROXY_PRINCIPAL_NAME  <br/> |
|Kennung:  <br/> |0x6625  <br/> |
|Eigenschaftentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Sonstiges  <br/> |
