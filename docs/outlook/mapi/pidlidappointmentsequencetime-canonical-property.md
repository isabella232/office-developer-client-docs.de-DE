# <a name="pidlidappointmentsequencetime-canonical-property"></a>Kanonische Pidlidappointmentsequencetime (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das Datum und die Uhrzeit der letzten Änderung dieser Eigenschaft an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidApptSeqTime  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Appointment  <br/> |
|Long-ID (Deckel):  <br/> |0x00008202  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Der Wert muss in koordinierter weltZeit (UTC) angegeben werden.
