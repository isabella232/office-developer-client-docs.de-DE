# <a name="pidlidtaskglobalid-canonical-property"></a>Kanonische PidLidTaskGlobalId-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Sucht eine vorhandene Aufgabe mithilfe einer GUID nach Erhalt einer Aufgabenantwort oder Aufgabenaktualisierung.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidTaskGlobalObjId  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008519  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Vorgang  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaft ist für nicht zugewiesene Vorgänge nicht festgelegt.
