# <a name="pidlidoldwhenstartwhole-canonical-property"></a>Kanonische Pidlidoldwhenstartwhole (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den ursprünglichen Wert der **dispidApptStartWhole** ([pidlidappointmentstartwhole (](pidlidappointmentstartwhole-canonical-property.md))-Eigenschaft vor einer Besprechungsaktualisierung an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidOldWhenStartWhole  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Meeting  <br/> |
|Long-ID (Deckel):  <br/> |0x00000029  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist nicht erforderlich.
