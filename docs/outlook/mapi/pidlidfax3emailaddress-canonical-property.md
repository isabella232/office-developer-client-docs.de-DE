# <a name="pidlidfax3emailaddress-canonical-property"></a>Kanonische PidLidFax3EmailAddress-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die E-Mail-Adresse der anderen Faxadresse des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFax3EmailAddress  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080D3  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaft sollte, falls vorhanden, einen benutzerlesbaren Anzeigenamen enthalten, gefolgt vom Zeichen "@" gefolgt von einer Faxnummer.
