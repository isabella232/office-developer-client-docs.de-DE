# <a name="pidlidyomifirstname-canonical-property"></a>Kanonische PidLidYomiFirstName-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die phonetische Aussprache des Vornamens des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidYomiFirstName  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x0000802C  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
