# <a name="pidnamelinecount-canonical-property"></a>Kanonische Pidnamelinecount (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Anzahl der Zeilen in der Datei an, die an das Dokument angefügt ist.
  
|||
|:-----|:-----|
|Angezeigte Namen:  <br/> |Keine  <br/> |
|Eigenschaftensatz:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Eigenschaftsname:  <br/> |LineCount  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Standard  <br/> |
