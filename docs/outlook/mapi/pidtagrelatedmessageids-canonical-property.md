# <a name="pidtagrelatedmessageids-canonical-property"></a>PidTagRelatedMessageIds (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Liste von Bezeichnern für Nachrichten, mit denen eine Nachricht verknüpft ist.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_RELATED_IPMS  <br/> |
|Kennung:  <br/> |0x002D  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |MAPI-Umschlag  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Die Bezeichner verwenden die gleichen spezifischen Konstruktionsregeln wie für die **PR_SEARCH_KEY** ([PidTagSearchKey](pidtagsearchkey-canonical-property.md)) -Eigenschaft.
