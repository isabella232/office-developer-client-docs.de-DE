# <a name="pidtagredirectionhistory-canonical-property"></a>Kanonische Pidtagredirectionhistory (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält Informationen zu der von einer zugestellten Nachricht abgedeckten Route.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_REDIRECTION_HISTORY  <br/> |
|Kennung:  <br/> |0x002C  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |MAPI-Umschlag  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft wird verwendet, wenn eine Nachricht von einem automatischen Agent automatisch weitergeleitet wird.
