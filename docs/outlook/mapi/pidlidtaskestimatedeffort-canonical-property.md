# <a name="pidlidtaskestimatedeffort-canonical-property"></a>Kanonische Pidlidtaskestimatedeffort (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Zeitdauer (in Minuten) an, die der Benutzer zum Ausführen einer Aufgabe erwartet.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidTaskEstimatedEffort  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Task  <br/> |
|Long-ID (Deckel):  <br/> |0x00008111  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Aufgabe  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Der Wert muss größer oder gleich 0 und kleiner als 0x5AE980DF (1.525.252.319) sein, wobei 480 Minuten ein Tag und 2400 Minuten eine Woche (acht Stunden an einem Arbeitstag und fünf Tage in einer Arbeitswoche) entsprechen.
