# <a name="pidnamemanager-canonical-property"></a>Kanonische PidNameManager-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Vorgesetzten der Datei an, die an das Dokument angefügt ist.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keines  <br/> |
|Eigenschaftensatz:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Eigenschaftenname:  <br/> |Manager  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Standard  <br/> |
