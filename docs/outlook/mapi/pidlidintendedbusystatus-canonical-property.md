# <a name="pidlidintendedbusystatus-canonical-property"></a>Kanonische PidLidIntendedBusyStatus-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Wert der **Eigenschaft dispidBusyStatus** ([PidLidBusyStatus](pidlidbusystatus-canonical-property.md)) für die Besprechung im Kalender des Organisators an, wenn die Besprechungsanfrage oder besprechungsaktualisierung gesendet wurde.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidIntendedBusyStatus  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008224  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Die zulässigen Werte dieser Eigenschaft sind mit denen für die Eigenschaft **dispidBusyStatus** identisch.
