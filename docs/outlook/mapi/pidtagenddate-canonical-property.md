# <a name="pidtagenddate-canonical-property"></a>PidTagEndDate (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält das Enddatum und die Endzeit eines Termins, wie von einer Planungsanwendung verwaltet. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_END_DATE  <br/> |
|Kennung:  <br/> |0x0061  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |MAPI-Umschlag  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Planungsanwendungen sollten beim Senden von Besprechungsanfragen sowohl die **PR_START_DATE** ([PidTagStartDate](pidtagstartdate-canonical-property.md)) als auch diese Eigenschaft festlegen.
