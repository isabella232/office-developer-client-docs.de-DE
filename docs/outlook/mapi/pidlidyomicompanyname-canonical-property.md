# <a name="pidlidyomicompanyname-canonical-property"></a>Kanonische PidLidYomiCompanyName-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die phonetische Aussprache des Firmennamens des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidYomiCompanyName  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x0000802E  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
