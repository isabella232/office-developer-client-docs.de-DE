# <a name="pidtagruleactions-canonical-property"></a>PidTagRuleActions (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Satz von Aktionen, die der Regel zugeordnet sind. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_RULE_ACTIONS  <br/> |
|Kennung:  <br/> |0x6680  <br/> |
|Datentyp:  <br/> |PT_ACTIONS  <br/> |
|Bereich:  <br/> |Serverseitige Regeln  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Die Aktionen werden als Regelaktion ausgedrückt, und der Eigenschaftswertpuffer enthält die Datenpufferstruktur der Regelaktion, die wie in [[MS-OXORULE]](https://msdn.microsoft.com/library/70ac9436-501e-43e2-9163-20d2b546b886%28Office.15%29.aspx)angegeben gepackt ist.
