# <a name="pidlidnotey-canonical-property"></a>Kanonische PidLidNoteY-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Abstand in Pixeln vom oberen Rand des Bildschirms an, in dem eine Benutzeroberfläche eine Notiz anzeigt.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidNoteY  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Note  <br/> |
|Long ID (LID):  <br/> |0x00008B05  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Kurznotiz  <br/> |
