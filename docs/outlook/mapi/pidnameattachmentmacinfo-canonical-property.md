# <a name="pidnameattachmentmacinfo-canonical-property"></a>PidNameAttachmentMacInfo (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen [RFC3282]-Anlagenwert, der aus geeigneten Header- und Ressourcen-Verzweigungsdaten besteht, die von Macintosh-basierten E-Mail-Clients verwendet werden.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keines  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Attachment  <br/> |
|Eigenschaftenname:  <br/> |AttachmentMacInfo  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Nachrichtenanlage  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Weitere Informationen finden Sie unter MS-OXCMAIL Section 2.2.4.2 Apple File Formats.
