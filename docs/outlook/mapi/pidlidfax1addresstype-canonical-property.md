# <a name="pidlidfax1addresstype-canonical-property"></a>Kanonische Pidlidfax1addresstype (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Adresstyp für die geschäftliche Faxnummer für einen Kontakt an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFax1AddrType  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long-ID (Deckel):  <br/> |0x000080B2  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft, sofern vorhanden, muss auf "FAX" festgelegt werden.
