# <a name="pidlidissilent-canonical-property"></a>Kanonische PidLidIsSilent-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, dass der Benutzer keinen Text in den Textkörper der Besprechungsantwort eingeschlossen hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |LID_IS_SILENT  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Meeting  <br/> |
|Long ID (LID):  <br/> |0x00000004  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
