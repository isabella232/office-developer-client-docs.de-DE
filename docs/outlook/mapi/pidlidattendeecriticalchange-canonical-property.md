# <a name="pidlidattendeecriticalchange-canonical-property"></a>Kanonische pidlidattendeecriticalchange (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das Datum und die Uhrzeit an, zu der das Besprechungs bezogene Objekt gesendet wurde.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |LID_ATTENDEE_CRITICAL_CHANGE  <br/> |
|Eigenschaftengruppe:  <br/> |PSETID_Meeting  <br/> |
|Lange ID (LID):  <br/> |0x00000001  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Hinweise

Der Wert muss in UTC (Coordinated Universal Time, koordinierte Weltzeit) angegeben werden.
