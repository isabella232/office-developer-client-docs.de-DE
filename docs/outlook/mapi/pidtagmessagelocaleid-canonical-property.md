# <a name="pidtagmessagelocaleid-canonical-property"></a>Kanonische Pidtagmessagelocaleid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Windows-LCID des Endbenutzers, der diese Nachricht erstellt hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_MESSAGE_LOCALE_ID  <br/> |
|Kennung:  <br/> |0x3FF1  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Sonstiges  <br/> |
