# <a name="pidtagfinderentryid-canonical-property"></a>PidTagFinderEntryId (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Eintragsbezeichner für den Ordner, in dem in der Regel Suchergebnisse erstellt werden.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_FINDER_ENTRYID  <br/> |
|Kennung:  <br/> |0x35E7  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |MAPI-Nachrichtenspeicher  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Der in dieser Eigenschaft enthaltene Eintragsbezeichner hat dasselbe Format wie die [ENTRYID-Struktur.](entryid.md)
