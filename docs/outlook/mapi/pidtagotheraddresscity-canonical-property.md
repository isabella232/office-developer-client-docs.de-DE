# <a name="pidtagotheraddresscity-canonical-property"></a>PidTagOtherAddressCity (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Namen des anderen Orts des E-Mail-Benutzers, z. B. die Stadt oder Stadt.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_OTHER_ADDRESS_CITY, PR_OTHER_ADDRESS_CITY_A, PR_OTHER_ADDRESS_CITY_W  <br/> |
|Kennung:  <br/> |0x3A5F  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaften werden vom Benutzer oder der Organisation des Benutzers definiert.
