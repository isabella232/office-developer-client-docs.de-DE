# <a name="pidlidtaskstatusoncomplete-canonical-property"></a>Kanonische PidLidTaskStatusOnComplete-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob der Aufgabenempfänger aufgefordert wurde, eine E-Mail-Nachrichtenaktualisierung zu senden, wenn er die zugewiesene Aufgabe abschließt.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidTaskSOC  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x00008119  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Aufgabe  <br/> |
