# <a name="pidlidemail2originalentryid-canonical-property"></a>PidLidEmail2OriginalEntryId (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die **EntryId** des Objekts an, das der zweiten E-Mail-Adresse entspricht. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidEmail2OriginalEntryID  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008095  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Der Wert dieser Eigenschaft muss entweder eine einmalige **EntryId** für diese elektronische Adresse oder ein gültiges Adressbuchobjekt **EntryId** sein.
