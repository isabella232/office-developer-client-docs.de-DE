# <a name="pidlidappointmentreplytime-canonical-property"></a>Kanonische PidLidAppointmentReplyTime-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das Datum und die Uhrzeit an, zu der der Teilnehmer auf eine empfangene Besprechungsanfrage oder besprechungsaktualisierung geantwortet hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidApptReplyTime  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008220  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Der Wert muss in koordinierter Weltzeit (COORDINATED Universal Time, UTC) angegeben werden.
