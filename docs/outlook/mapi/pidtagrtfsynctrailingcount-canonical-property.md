# <a name="pidtagrtfsynctrailingcount-canonical-property"></a>Kanonische Pidtagrtfsynctrailingcount (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Anzahl der ignorierten Zeichen, die nach den signifikanten Zeichen der Nachricht angezeigt werden.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_RTF_SYNC_TRAILING_COUNT  <br/> |
|Kennung:  <br/> |0x1011  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |MAPI-Nachricht  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist eine RFT-Eigenschaft (Rich Text Format). Diese Eigenschaften werden von der [RTFSync](rtfsync.md) -Funktion verwendet und sind nicht für die direkte Verwendung durch Clientanwendungen vorgesehen.
