# <a name="pidnamelastsavedatetime-canonical-property"></a>PidNameLastSaveDateTime (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Zeitpunkt an, zu dem die Datei zuletzt gespeichert wurde.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keines  <br/> |
|Eigenschaftensatz:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Eigenschaftenname:  <br/> |LastSaveDtm  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Standard  <br/> |
