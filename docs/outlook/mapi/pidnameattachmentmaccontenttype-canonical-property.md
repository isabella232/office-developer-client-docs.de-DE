# <a name="pidnameattachmentmaccontenttype-canonical-property"></a>PidNameAttachmentMacContentType (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen [RFC3282]-Headerfeldwert für Anlagen, die von Macintosh-basierten E-Mail-Clients verwendet werden.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keines  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Attachment  <br/> |
|Eigenschaftenname:  <br/> |AttachmentMacContentType  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Nachrichtenanlage  <br/> |
