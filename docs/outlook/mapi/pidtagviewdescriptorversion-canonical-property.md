# <a name="pidtagviewdescriptorversion-canonical-property"></a>Kanonische Pidtagviewdescriptorversion (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Version eines Ansichts Deskriptors an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_VD_VERSION  <br/> |
|Kennung:  <br/> |0x7007  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Sonstiges  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Dieser Wert der Eigenschaft muss 8 sein.
