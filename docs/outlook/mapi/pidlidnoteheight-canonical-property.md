# <a name="pidlidnoteheight-canonical-property"></a>Kanonische PidLidNoteHeight-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Höhe des sichtbaren Meldungsfensters in Pixel an. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidNoteHeight  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Note  <br/> |
|Long ID (LID):  <br/> |0x00008B03  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Kurznotiz  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Dieser Wert muss größer als Null sein.
