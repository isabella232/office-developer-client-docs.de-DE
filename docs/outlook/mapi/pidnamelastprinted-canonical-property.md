# <a name="pidnamelastprinted-canonical-property"></a>Kanonische Pidnamelastprinted (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Zeitpunkt an, zu dem die Datei zuletzt gedruckt wurde.
  
|||
|:-----|:-----|
|Angezeigte Namen:  <br/> |Keine  <br/> |
|Eigenschaftensatz:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Eigenschaftsname:  <br/> |LastPrinted  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Standard  <br/> |
