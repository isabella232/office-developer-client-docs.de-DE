# <a name="pidtagimapcachedmsgsize-canonical-property"></a>Kanonische Pidtagimapcachedmsgsize (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Größe der zwischengespeicherten Internet Message Access-Protokollnachricht.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |Keine  <br/> |
|Kennung:  <br/> |0x10F0  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Nachrichten Ordner umbenennen  <br/> |
