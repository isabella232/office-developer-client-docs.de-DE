# <a name="pidtagotheraddresscountry-canonical-property"></a>Kanonische Pidtagotheraddresscountry (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält das andere Land/eine Region des e-Mail-Benutzers.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_OTHER_ADDRESS_COUNTRY, PR_OTHER_ADDRESS_COUNTRY_A, PR_OTHER_ADDRESS_COUNTRY_W  <br/> |
|Kennung:  <br/> |0x3A60  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Adressbuch  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften werden vom Benutzer oder der Organisation des Benutzers definiert.
