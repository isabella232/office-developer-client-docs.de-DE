# <a name="pidlidverbstream-canonical-property"></a>Kanonische PidLidVerbStream-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, welche Abstimmungsantworten der Benutzer als Reaktion auf die Nachricht vornehmen kann.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidVerbStream  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008520  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Laufzeitkonfiguration  <br/> |
