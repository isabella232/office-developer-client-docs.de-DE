# <a name="pidlidclipstart-canonical-property"></a>Kanonische pidlidclipstart (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das Startdatum und die Uhrzeit des Ereignisses in koordinierten universellen Zeiten (UTC) für einzelne Instanzen von Kalenderobjekten an und gibt Mitternacht am Datum der ersten Instanz in UTC für eine wiederkehrende Reihe an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidClipStart  <br/> |
|Eigenschaftengruppe:  <br/> |PSETID_Appointment  <br/> |
|Lange ID (LID):  <br/> |0x00008235  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Kalender  <br/> |
