# <a name="pidtagexpirytime-canonical-property"></a>PidTagExpiryTime (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält das Datum und die Uhrzeit, zu der das Messagingsystem den Inhalt einer Nachricht ungültig macht. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_EXPIRY_TIME  <br/> |
|Kennung:  <br/> |0x0015  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |MAPI-Umschlag  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaft wird verwendet, um das Messagingsystem bei der Behandlung von übermittelten zwischenpersönlichen Nachrichten zu leiten.
