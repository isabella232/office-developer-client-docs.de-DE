# <a name="pidtagnickname-canonical-property"></a>Kanonische Pidtagnickname (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Spitznamen des Kontakts.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_NICKNAME, PR_NICKNAME_A, PR_NICKNAME_W  <br/> |
|Kennung:  <br/> |0x3A4F  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften sind optional. Ihre Inhalte werden vom Benutzerdefiniert.
