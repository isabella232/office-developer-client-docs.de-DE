# <a name="pidtagconflictentryid-canonical-property"></a>Kanonische Pidtagconflictentryid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Eintrags-e-Mail der Konflikt Lösungs Meldung.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_CONFLICT_ENTRYID  <br/> |
|Kennung:  <br/> |0x3FF0  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |ICS (Incremental Change Synchronization)  <br/> |
