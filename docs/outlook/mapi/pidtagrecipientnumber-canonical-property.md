# <a name="pidtagrecipientnumber-canonical-property"></a>Kanonische Pidtagrecipientnumber (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Stellt die Empfängernummer eines Empfängers in der Nachricht dar.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_RECIPIENT_NUMBER  <br/> |
|Kennung:  <br/> |0x6662  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Lese-/Schreibzugriff für Exchange-nachRichten  <br/> |
