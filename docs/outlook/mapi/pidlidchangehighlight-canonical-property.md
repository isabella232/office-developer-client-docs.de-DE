# <a name="pidlidchangehighlight-canonical-property"></a>Kanonische PidLidChangeHighlight-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt ein Bitfeld an, das angibt, wie sich das Besprechungsobjekt geändert hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidChangeHighlight  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008204  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaft ist nicht erforderlich. Die einzelnen Kennzeichen, die festgelegt werden können, sind in [[MS-OXOCAL]](https://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx)aufgeführt.
