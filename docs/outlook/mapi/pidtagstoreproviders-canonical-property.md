# <a name="pidtagstoreproviders-canonical-property"></a>PidTagStoreProviders (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Liste der Bezeichner von Nachrichtenspeicheranbietern im aktuellen Profil.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_STORE_PROVIDERS  <br/> |
|Kennung:  <br/> |0x3D00  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |MAPI-Profil  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Verwenden Sie diese Eigenschaft nicht. Sie ist für die Verwendung durch mapi reserviert.
