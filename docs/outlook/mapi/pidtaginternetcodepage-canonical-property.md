# <a name="pidtaginternetcodepage-canonical-property"></a>Kanonische PidTagInternetCodepage-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Codepage an, die für **PR_BODY** -[PidTagBody](pidtagbody-canonical-property.md)) oder **PR_BODY_HTML** ([PidTagBodyHtml](pidtagbodyhtml-canonical-property.md)) -Eigenschaften verwendet wird.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_INTERNET_CPID  <br/> |
|Kennung:  <br/> |0x3FDE  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Sonstiges  <br/> |
