# <a name="pidlidautofilllocation-canonical-property"></a>Kanonische PidLidAutoFillLocation-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, dass der Wert der **dispidLocation** ([PidLidLocation](pidlidlocation-canonical-property.md)) -Eigenschaft auf die **PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)) -Eigenschaft aus dem RecipientRow festgelegt ist, die eine Ressource darstellt.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidAutoFillLocation  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x0000823A  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
