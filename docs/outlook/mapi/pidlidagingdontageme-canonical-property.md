# <a name="pidlidagingdontageme-canonical-property"></a>Kanonische pidlidagingdontageme (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Wert true an, wenn das Element nicht automatisch archiviert wird.
