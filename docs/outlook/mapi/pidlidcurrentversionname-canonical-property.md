# <a name="pidlidcurrentversionname-canonical-property"></a>Kanonische Pidlidcurrentversionname (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Zeichenfolge, die die Haupt-und Nebenversionsnummer der Outlook-Anwendung für ein Element angibt.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidCurrentVersionName  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long-ID (Deckel):  <br/> |0x00008554  <br/> |
|Datentyp:  <br/> |Noch nicht festgelegt  <br/> |
|Bereich:  <br/> |Allgemeine Nachrichtenübermittlung  <br/> |
