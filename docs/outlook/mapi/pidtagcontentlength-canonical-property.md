# <a name="pidtagcontentlength-canonical-property"></a>Kanonische Pidtagcontentlength (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Nachrichtenlänge in Byte, die an eine Clientanwendung oder einen Dienstanbieter übergeben wird, um zu bestimmen, ob eine Nachricht dieser Länge zugestellt werden kann. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_CONTENT_LENGTH  <br/> |
|Kennung:  <br/> |0x0009  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Exchange  <br/> |
