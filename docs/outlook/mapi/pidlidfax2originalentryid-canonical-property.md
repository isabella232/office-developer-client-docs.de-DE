# <a name="pidlidfax2originalentryid-canonical-property"></a>PidLidFax2OriginalEntryId (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die ursprüngliche EntryID der Homefaxadresse des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFax2OriginalEntryID  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Lange ID (LID):  <br/> |0x000080C5  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft muss, falls vorhanden, die one-off EntryId angeben, die dieser Faxadresse entspricht.
