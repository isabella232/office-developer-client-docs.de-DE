# <a name="pidtagkeyword-canonical-property"></a>Kanonische PidTagKeyword-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält ein Schlüsselwort, das den Empfänger für den Systemadministrator des Empfängers identifiziert.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_KEYWORD, PR_KEYWORD_A, PR_KEYWORD_W  <br/> |
|Kennung:  <br/> |0x3A0B  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaften bieten Identifikations- und Zugriffsinformationen für einen Empfänger. Sie werden vom Empfänger und seiner Organisation definiert.
