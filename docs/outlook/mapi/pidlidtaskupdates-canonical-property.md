# <a name="pidlidtaskupdates-canonical-property"></a>Kanonische PidLidTaskUpdates-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob der Aufgabenempfänger aufgefordert wurde, eine Aufgabenaktualisierung zu senden, wenn sich die zugewiesene Aufgabe ändert.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidTaskUpdates  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x0000811B  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Aufgabe  <br/> |
