# <a name="pidlidfileunderlist-canonical-property"></a>Kanonische PidLidFileUnderList-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt eine Liste möglicher Werte für die **Eigenschaft dispidFileUnderId** ([PidLidFileUnderId](pidlidfileunderid-canonical-property.md)) an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFileUnderList  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008026  <br/> |
|Datentyp:  <br/> |PT_MV_LONG  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Jeder Wert in der Mehrwerteigenschaft muss einer der zulässigen Werte für **dispidFileUnderId** sein, der in [[MS-OXOCNTC]](https://msdn.microsoft.com/library/9b636532-9150-4836-9635-9c9b756c9ccf%28Office.15%29.aspx)angegeben ist.
