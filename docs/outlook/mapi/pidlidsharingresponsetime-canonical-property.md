# <a name="pidlidsharingresponsetime-canonical-property"></a>Kanonische Pidlidsharingresponsetime (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Zeit an, zu der der Empfänger der Freigabeanforderung eine Freigabeantwort gesendet hat. Dies ist eine Eigenschaft einer Freigabenachricht.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidSharingResponseTime  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Sharing  <br/> |
|Long-ID (Deckel):  <br/> |0x00008A28  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Freigabe  <br/> |
