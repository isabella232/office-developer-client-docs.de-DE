# <a name="pidtagspamjunksenders-canonical-property"></a>PidTagSpamJunkSenders (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine durch Semikolons getrennte Liste von E-Mail-Adressen und Domänen, die blockierte Absender umfassen.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_SPAM_JUNK_SENDERS_W  <br/> |
|Long ID (LID):  <br/> |0x041A  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Spam  <br/> |
