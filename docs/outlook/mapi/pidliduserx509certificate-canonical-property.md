# <a name="pidliduserx509certificate-canonical-property"></a>Kanonische PidLidUserX509Certificate-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält X.509 Version 3-Sicherheitszertifikate für einen Messagingbenutzer.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidEMSAbX509Cert  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080D9  <br/> |
|Datentyp:  <br/> |PT_MV_BINARY  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
