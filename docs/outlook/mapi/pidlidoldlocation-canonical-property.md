# <a name="pidlidoldlocation-canonical-property"></a>Kanonische PidLidOldLocation-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den ursprünglichen Wert der **dispidLocation** -[PidLidLocation](pidlidlocation-canonical-property.md)) -Eigenschaft vor einer Besprechungsaktualisierung an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidOldLocation  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Meeting  <br/> |
|Long ID (LID):  <br/> |0x00000028  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaft ist nicht erforderlich.
