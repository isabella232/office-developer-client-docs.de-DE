# <a name="pidlidownercriticalchange-canonical-property"></a>PidLidOwnerCriticalChange (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das Datum und die Uhrzeit an, zu der eine Besprechungsanfrage vom Organisator gesendet wurde.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |LID_OWNER_CRITICAL_CHANGE  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Meeting  <br/> |
|Long ID (LID):  <br/> |0x0000001A  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Der Wert muss in koordinierter Weltzeit (COORDINATED Universal Time, UTC) angegeben werden.
