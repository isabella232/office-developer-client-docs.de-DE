# <a name="pidlidlogflags-canonical-property"></a>Kanonische PidLidLogFlags-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält Metadaten zum Journal.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidLogFlags  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Log  <br/> |
|Long ID (LID):  <br/> |0x0000870C  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Journal  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Das Bitfeld, das Metadaten zum Journal enthält, muss null oder "0x40000000" sein.
