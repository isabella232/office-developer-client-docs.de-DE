# <a name="pidnamexsharingremoteuid-canonical-property"></a>Kanonische PidNameXSharingRemoteUid-Eigenschaft

|||

|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Freigabe  <br/> |
