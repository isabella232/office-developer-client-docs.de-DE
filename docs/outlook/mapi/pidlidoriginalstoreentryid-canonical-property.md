# <a name="pidlidoriginalstoreentryid-canonical-property"></a>Kanonische PidLidOriginalStoreEntryId-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Eintrags-ID des Speichers des Delegators an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidOrigStoreEid  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008237  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaft sollte für Besprechungsobjekte festgelegt werden, die von einem Delegaten erstellt oder aktualisiert wurden.
