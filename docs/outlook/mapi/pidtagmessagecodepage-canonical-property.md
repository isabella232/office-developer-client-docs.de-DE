# <a name="pidtagmessagecodepage-canonical-property"></a>Kanonische PidTagMessageCodepage-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Codepage, die für die Nachricht verwendet wird.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_MESSAGE_CODEPAGE  <br/> |
|Kennung:  <br/> |0x3FFD  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Standard  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Die Codepage des Ordnerobjekts wird verwendet, wenn diese Eigenschaft auf Null (0) festgelegt ist.
