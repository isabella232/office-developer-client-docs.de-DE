# <a name="pidlidcontactuserfield2-canonical-property"></a>PidLidContactUserField2 (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das zweite Feld des Kontakts an, das für verschiedene Zwecke für den Kontakt vorgesehen ist.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidContactUserField2  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008050  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft enthält Text, der nichts mit einer anderen kontaktspezifischen Eigenschaft zu tun hat.
