# <a name="pidtaglastverbexecutiontime-canonical-property"></a>Kanonische Pidtaglastverbexecutiontime (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Uhrzeit, zu der das letzte Verb ausgeführt wurde.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_LAST_VERB_EXECUTION_TIME  <br/> |
|Kennung:  <br/> |0x1082  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Verlauf  <br/> |
