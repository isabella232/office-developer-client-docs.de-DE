# <a name="pidlidsharingconfigurationurl-canonical-property"></a>Kanonische PidLidSharingConfigurationUrl-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Wird als Eigenschaft einer Freigabenachricht festgelegt.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidSharingConfigUrl  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Sharing  <br/> |
|Long ID (LID):  <br/> |0x00008A24  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Freigabe  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Der Wert dieser Eigenschaft muss eine leere Zeichenfolge sein.
