# <a name="pidtagcompanyname-canonical-property"></a>PidTagCompanyName (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Firmennamen des Empfängers. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_COMPANY_NAME, PR_COMPANY_NAME_A, PR_COMPANY_NAME_W  <br/> |
|Kennung:  <br/> |0x3A16  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaften bieten Identifikations- und Zugriffsinformationen zu einem Empfänger. Sie werden vom Empfänger und der Organisation des Empfängers definiert.
