# <a name="pidtagspamtrustedsenders-canonical-property"></a>PidTagSpamTrustedSenders (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine durch Semikolons getrennte Liste von E-Mail-Adressen und Domänen, die aus vertrauenswürdigen Absendern bestehen.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_SPAM_TRUSTED_SENDERS_W  <br/> |
|Long ID (LID):  <br/> |0x0418  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Spam  <br/> |
