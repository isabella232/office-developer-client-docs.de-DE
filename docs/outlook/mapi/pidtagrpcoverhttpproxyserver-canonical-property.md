# <a name="pidtagrpcoverhttpproxyserver-canonical-property"></a>Kanonische PidTagRpcOverHttpProxyServer-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die URL des Proxyservers, der zum Herstellen einer Verbindung zwischen Microsoft Outlook und Microsoft Exchange Server verwendet wird, ohne das HTTP-Protokoll oder das HTTPS-Protokoll (Hypertext Transfer Protocol Secure), das vor der URL vorkommt.
  
## 

|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_ROH_PROXY_SERVER  <br/> |
|Kennung:  <br/> |0x6622  <br/> |
|Eigenschafts:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Sonstiges  <br/> |
