|Element|Code|
|:-----|:-----|
|Kennung:  <br/> |0x0205  <br/> |
|Eigenschaftentyp:  <br/> |PT_UNICODE|SECURE_FLAG  <br/> |
|Eigenschaftstag:  <br/> |0x0205801F  <br/> |
|Zugriff:  <br/> |Schreibgeschützt  <br/> |
