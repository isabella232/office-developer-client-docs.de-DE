# <a name="functions-outlook-exported-apis"></a>Funktionen (Outlook-exportierte APIs)

In diesem Abschnitt werden die Funktionen beschrieben, die Outlook exportiert.
  
|**Funktion**|**Beschreibung**|
|:-----|:-----|
|[HrCreateApptRebaser](hrcreateapptrebaser.md) <br/> |Initialisiert ein [IOlkApptRebaser](iolkapptrebaser.md) -Objekt für die Verwendung beim erneuten Aufsetzen von Terminen in Outlook-Kalendern.  <br/> |
|[HrProcessConvActionForSentItem](hrprocessconvactionforsentitem.md) <br/> |Führt die Kategorisierung nach dem Senden für ein e-Mail-Element basierend auf dessen [PidTagConversationId](https://msdn.microsoft.com/library/f8e4a5fa-cb73-4eca-b174-72e1fda821a6%28Office.15%29.aspx)aus.  <br/> |
|[RebaseTaskComplete](rebasetaskcomplete.md) <br/> |Meldet den Abschluss für die erneute Erstellung von Terminen.  <br/> |
|[RebaseTaskProgress](rebasetaskprogress.md) <br/> |Meldet den Fortschritt für die Aufzählung und die Begründung von Terminen.  <br/> |
