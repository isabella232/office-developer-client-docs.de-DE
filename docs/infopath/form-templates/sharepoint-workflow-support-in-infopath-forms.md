# <a name="sharepoint-workflow-support-in-infopath-forms"></a>Unterstützung von SharePoint-Workflow in InfoPath-Formularen

Der Dokumentworkflow ist ein Feature von Microsoft SharePoint Server 2010 und Microsoft SharePoint Foundation 2010, mit dem Sie einen Workflow definieren und in einer Dokumentbibliothek verwenden können. Szenarien für integrierten Workflow, wie z. B. das Routen eines Dokuments zur Genehmigung oder das Sammeln von Feedback zu einem Dokument, sind häufig verwendete Workflowvorlagen, die an Ihre speziellen Anforderungen angepasst werden können. Jedem Office-Dokument kann ein Workflow zugeordnet werden, aber Microsoft InfoPath-Formularvorlagen können auch so entworfen werden, dass Workflowinformationen in die InfoPath-Formularvorlage integriert werden. 
  
Informationen zum Entwerfen von InfoPath-Formularen für Workflows finden Sie unter [InfoPath Forms for Workflows](https://msdn.microsoft.com/library/f2b0423e-22d7-485f-a723-19fa68759ef3%28Office.15%29.aspx) in der [sharepoint Server 2010-allgemeinen Referenz](https://msdn.microsoft.com/library/b3bf6083-997f-48c3-9ea3-e351439699ba%28Office.15%29.aspx) Dokumentation.
