## <a name="deploying-form-templates-that-require-full-trust"></a><span data-ttu-id="e40d4-119">Bereitstellen von Formularvorlagen, die volle Vertrauenswürdigkeit erfordern</span><span class="sxs-lookup"><span data-stu-id="e40d4-119">Deploying Form Templates That Require Full Trust</span></span>

<span data-ttu-id="e40d4-p105">Wenn im Formularcode für die Formularvorlage InfoPath-Objektmodellmember verwendet werden, für die volle Vertrauenswürdigkeit erforderlich ist, oder wenn in der Formularvorlage Features verwendet werden, für die volle Vertrauenswürdigkeit erforderlich ist, müssen Sie die Formularvorlagendatei (XSN) digital mit einem Codesignierungszertifikat von einem vertrauenswürdigen Herausgeber signieren.
