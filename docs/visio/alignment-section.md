# <a name="alignment-section"></a>Abschnitt "Alignment"

Gibt die Ausrichtung einer Form in Bezug auf die Führungslinie oder den Führungspunkt an, an die die Form geklebt ist. Der **Abschnitt "Ausrichtung"** wird nur für Shapes angezeigt, die an Führungslinien geklebt sind.
