# <a name="image-properties-section"></a>Abschnitt "Image Properties"

Enthält die Werte Gamma, Helligkeit, Kontrast, Verwischen, Schärfen und Entrauschen für Bitmaps (aus Pixeln bestehende Grafiken) und wird nur für solche Objekte im ShapeSheet-Fenster angezeigt.
