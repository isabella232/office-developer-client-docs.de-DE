# <a name="imgoffsetx-cell-foreign-image-info-section"></a>Zelle "ImgOffsetX" (Abschnitt "Foreign Image Info")

Bestimmt den Abstand, den das Objekt horizontal vom Ursprung des Rahmens des Objekts entfernt ist. Der Standardwert ist 0. Durch Bearbeitung des Objekts mit dem Zuschneidetool wird dieser Wert geändert. 
  
## <a name="remarks"></a>HinwBemerkungeneise

Um einen Verweis auf die Zelle "ImgOffsetX" anhand des Namens aus einer anderen Formel oder aus einem Programm mithilfe der **CellsU-Eigenschaft** abzurufen, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
| Zellenname:  <br/> | ImgOffsetX  <br/> |
   
Um einen Verweis auf die Zelle "ImgOffsetX" anhand des Indexes eines Programms abzurufen, verwenden Sie die **CellsSRC-Eigenschaft** mit den folgenden Argumenten: 
  
|||
|:-----|:-----|
| Abschnittsindex:  <br/> |**visSectionObject** <br/> |
| Zeilenindex:  <br/> |**visRowForeign** <br/> |
| Zellenindex:  <br/> |**visFrgnImgOffsetX** <br/> |
