# <a name="glowcolor-cell-additional-effect-properties-section"></a>Zelle "GlowColor" (Abschnitt "Additional Effect Properties")

Bestimmt die Farbe, die für den Strich des auf eine Form angewendeten externen Leuchtens als RGB- oder Designwert verwendet wird.
  
## <a name="remarks"></a>HinwBemerkungeneise

Um einen Verweis auf die **GlowColor-Zelle** anhand des Namens aus einer anderen Formel, anhand des Werts des **N-Attributs** eines **Cell-Elements** oder eines Programms mithilfe der **CellsU-Eigenschaft** abzurufen, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
| Zellenname:  <br/> | GlowColor  <br/> |
   
Um einen Verweis auf die **GlowColor-Zelle** anhand des Indexes eines Programms abzurufen, verwenden Sie die **CellsSRC-Eigenschaft** mit den folgenden Argumenten: 
  
|||
|:-----|:-----|
| Abschnittsindex:  <br/> |**visSectionObject** <br/> |
| Zeilenindex:  <br/> |**visRowOtherEffectProperties** <br/> |
| Zellenindex:  <br/> |**visGlowColor** <br/> |
